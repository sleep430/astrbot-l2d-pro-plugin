# L2D Pro v1.1.1

## 新增

- 新增 **设置 → 模型 → 模型参数** 页面。
- 自动枚举当前 Live2D 模型的全部 Cubism 参数、默认值与取值范围。
- 支持按参数 ID 搜索、滑杆/数值框实时调整。
- 支持单项恢复和全部恢复，恢复后重新交由动作、表情、视线与口型驱动。

## 技术改进

- 增加设置窗口与主窗口之间的参数快照、实时覆盖 IPC。
- 参数覆盖在每帧动画计算后应用，避免被动作或表情覆盖。
- 参数值会按模型声明的最小值和最大值自动裁剪。

## 验证

- `pnpm run typecheck`
- `pnpm run lint`
- `pnpm exec vite build`
