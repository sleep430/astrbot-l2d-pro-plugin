# L2D Pro v1.1.0 更新内容

## 重点更新

- 更新桌面端自动更新源到 `sleep430/astrbot-l2d-pro-plugin`，避免后续拉取原桌面端发布包把改版覆盖掉。
- 新增 Live2DViewerEX 风格 `.lpk` 导入兼容，导入模型时可以直接尝试选择 `.lpk` 包。
- 新增桌面点击交互配置，支持按 HitArea、头部、身体等区域触发表情或动作。
- 新增点击交互、动作和表情测试入口，方便确认模型资源是否能正常触发。
- 新增表情自动回归配置，可设置触发后 x 秒回到默认表情或指定表情。
- 优化表情回归过渡，减少硬切造成的僵硬感。
- 调整前端视觉方向，设置界面向浅色半透明、漂浮式、简约风格靠拢。

## 本次刷新修复

- 修复设置页模型配置桥接缺失，确保模型配置的加载、保存、删除和初始化入口可用。
- 加固 Electron 窗口导航处理，外部链接会交给系统浏览器打开，避免桌面端窗口被直接跳转到外部网页。
- release 工作流会重新构建并覆盖 `v1.1.0` 资产，不新增版本号。
- 修复 GitHub smoke 检查环境，保留 Windows Server 2022 并使用 Node 20 运行跨平台验证。

## 使用说明

- 如果模型包含标准动作资源，点击交互可以绑定动作或表情。
- 如果模型只有表情没有交互动作，点击交互会退化为表情触发。
- Live2DViewerEX 的 `VarFloats` 等高级参数逻辑暂不处理。
- `.lpk` 兼容属于尽力适配，特殊加密、特殊脚本或非标准资源组织的模型可能仍需要手动整理。
